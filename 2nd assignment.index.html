ALL TEXT FORMATTING TAGS OF HTML

HTML TEXT FORMATING TAGS Meanings
<b> Bold tag The bold tag is used to darken our text in HTML. Generally styling in html is gradually moving away, thus developers are advised to use CSS
<strong> </strong>Strong tag The strong tag is used by the brower to locate important aspect of the your web page or where to lay emphasis on by darkening the text ramped inside it.
<i> Italic tag The <i> tag is used to italicise the text. It opens with <i> and ends with </i> tag.
<em> The <em> tag is used to emphasize the text, with added semantic importance. It opens with <em> and ends with </em> tag.
<mark> Marked text The <mark> tag is used to to highlight a text in HTML. It opens with <mark> and ends with </mark> tag.
<small> Smaller text The <small> element is used to make the text smaller. The text that needs to be displayed smaller should be written inside <small> and </small> tag.
<del> deleted text The <del> element is used to through the text marking the part as deleted. It also has an opening and a closing tag.
<ins> inserted text The <ins> element is used to underline a text marking the part as inserted or added. It also has an opening and a closing tag.
<sub> Subscript text The <sub> HTML element specifies inline text which should be displayed as subscript for solely typographical reasons. Subscripts are typically rendered with a lowered baseline using smaller text.
<sup> Superscript text The <sup> HTML element specifies inline text which is to be displayed as superscript for solely typographical reasons. Superscripts are usually rendered with a raised baseline using smaller text
